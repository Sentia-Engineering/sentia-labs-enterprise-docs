---
title: "List integrations"
api: "GET /integrations"
description: "List all connected integrations in this workspace with their sync state and recent activity."
---

Returns all integrations connected to the workspace, including auth status, sync state, and recent activity.

## Request

```bash
curl -X GET https://api.sentialabs.ai/integrations \
  -H "Authorization: Bearer $SENTIA_TOKEN" \
  -H "x-workspace-id: $WORKSPACE_ID"
```

## Query parameters

| Parameter | Notes |
| --- | --- |
| `category` | Filter by category: `analytics`, `qualitative`, `workflow`, `documentation`, `experimentation`, `ship_detection` |
| `status` | Filter by status: `connected`, `syncing`, `stale`, `error`, `disconnected` |

## Provider list

For the full list of supported providers, see [Integrations overview](/integrations/overview).

## Sync state semantics

| Status | Meaning |
| --- | --- |
| `connected` | Healthy; last sync succeeded recently |
| `syncing` | Currently mid-sync (long-running first syncs especially) |
| `stale` | Connected but last sync is older than 2× normal cadence |
| `error` | Last 3 sync attempts failed; investigation needed |
| `disconnected` | Credentials revoked or OAuth flow needs re-auth |

## Errors

| Status | Code | Cause |
| --- | --- | --- |
| 401 | `invalid_token` | Token missing or invalid |
| 403 | `workspace_access_denied` | Token doesn't grant access to this workspace |
