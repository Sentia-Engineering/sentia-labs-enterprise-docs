Sentia is built keyboard-first. Power users navigate almost entirely without the mouse. Shortcuts are registered via a singleton registry so they're consistent everywhere.

Press `?` in the app to open the live shortcut overlay.

## Global navigation

| Shortcut | Action |
| --- | --- |
| `S` | Simulations |
| `P` | Populations |
| `D` | Decisions |
| `I` | Integrations |
| `C` | Calibration |
| `⌘,` | Settings |
| `⌘B` | Toggle sidebar |
| `⌘⇧N` | Open notifications |
| `?` | Keyboard shortcuts overlay |
| `⌘K` | Command palette (search everything) |

## Lists and tables

| Shortcut | Action |
| --- | --- |
| `J` | Next row |
| `K` | Previous row |
| `Enter` | Open selected row or expand agent |
| `/` | Focus search |
| `1`–`6` | Filter by status (varies by list) |

| Shortcut | Action |
| --- | --- |

## Accessibility

All shortcuts respect `prefers-reduced-motion` — animations are disabled when set. Focus rings are always visible on the focused element.

For screen reader users:

- Skip-to-main link is always present (focus on page load)
- Focus moves to `<main>` on route change
- Agent counter updates announce via `aria-live` every 10 agents
- Tab panels are linked via `aria-controls` / `aria-labelledby`

## Customizing shortcuts

Shortcuts are not currently customizable. Submit feature requests to [support@sentialabs.ai](mailto:support@sentialabs.ai) for shortcuts that would help your team's workflow.

## Common workflows in keystrokes

Some flows users have memorized:

| Workflow | Keys |
| --- | --- |
| Run a new simulation from scratch | `N` → fill wizard → `⌘⇧L` |
| Open the most recent simulation and export | `S` → `Enter` → `⌘E` |
| Switch from results to populations | `P` (from anywhere) |
| Search for a specific decision | `D` → `/` → type → `Enter` |
| Run validation on a population | `P` → `J/K` to select → `Enter` → `⌘V` |
