---
title: "Keyboard shortcuts"
description: "The dashboard shortcuts available in the current Sentia app."
---

Press `?` in the app to open the live shortcut overlay. `⌘` maps to Command on macOS and Control on Windows/Linux where the browser supports it.

## Global

| Shortcut | Action |
| --- | --- |
| `⌘K` | Command palette |
| `⌘L` | Ask Sentia AI |
| `⌘⇧A` | Toggle AI Co-Pilot |
| `⌘⇧N` | Open notifications |
| `⌘B` | Toggle sidebar |
| `⌘,` | Settings |
| `?` | Shortcut overlay |

## Navigation

| Shortcut | Action |
| --- | --- |
| `O` | Overview |
| `D` | Design Agent |
| `S` | Simulations |
| `P` | Populations |
| `R` | Resource Library |
| `I` | Settings -> Integrations |

## Simulations

| Shortcut | Action |
| --- | --- |
| `N` | New simulation |
| `⌘Enter` | Advance to next wizard step |
| `⌘⌫` | Go back one wizard step |
| `Esc` | Exit or close the active overlay |

## Lists and tables

| Shortcut | Action |
| --- | --- |
| `/` | Focus search where available |
| `J` | Next row where list navigation is enabled |
| `K` | Previous row where list navigation is enabled |
| `Enter` | Open the selected row or item |

## Accessibility

Keyboard support is intended to keep core workflows usable without a mouse. Focus rings remain visible, route changes move focus to the main content area, and live simulation counters are announced where supported.

Shortcuts are not currently customizable. Send workflow-specific shortcut requests to [hello@sentialabs.io](mailto:hello@sentialabs.io).
